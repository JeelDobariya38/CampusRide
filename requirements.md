## High Level Overview (Core Functionality)

- Home/Landing Page for app itself. where we will list out the feature we care to provide. basically, a whole bunch of buzz word that attracts users.

- Login/SignUp Page for user authentication, it should collect user data (as in school-email & password) and send it to backend and then, it will be backend responsibility for validating the data, and if everything goes fine, backend will login the users & also will create new user account in case of signup.

- User Dashboard for giving (the current login-in user) information about the current realtime situation, as in how many rides are available. and other essential information

- User Profile Page for user profile overview. basically, here we will give user a way to customize profile and some bunch of user settings that make thier profile stand out.
