## Quick Getting Started (Local Development)

### Clone The Repository

```
git clone https://github.com/JeelDobariya38/CampusRide.git
```

### Setup Backend

```bash
cd backend
npm install
npm run dev
```

### Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
